# タイトル: 従業員情報に「趣味・嗜好」入力欄を追加

## 要約: 
従業員情報画面に「趣味・嗜好」の入力欄を追加し、従業員が各自の趣味や興味を登録できるようにします。これにより、人事部や総務部などの福利厚生担当部署が従業員の興味関心を把握しやすくなり、福利厚生企画の効率化を図ります。趣味・嗜好に関する情報が散在している現状を改善し、企画担当者の情報収集時間を短縮することを目的とします。

## ユーザーストーリー: 
- **[誰が]**: 全従業員、人事部、総務部、福利厚生を担当する部署
- **[何のために]**: 福利厚生の企画立案を効率化し、従業員のニーズに合った企画を実施するため
- **[何ができる]**: 従業員は自分の趣味・嗜好を登録でき、管理側は従業員の趣味・嗜好を一覧・検索できる

3.  **CSVファイルのヘッダーについて:**

    ```python
    def add_employee():
        if not os.path.exists(EMPLOYEE_CSV):
            with open(EMPLOYEE_CSV, 'w', newline='', encoding='utf-8') as f:
                writer = csv.writer(f)
                writer.writerow(['ID', '名前', '部署', '所属組織ID', '趣味・嗜好']) # ヘッダー行の追加
        emp_id = input('従業員ID: ')
        name = input('名前: ')
        dept = input('部署: ')
        org_id = input('所属組織ID: ')
        hobby = input('趣味・嗜好: ') # 追加
        with open(EMPLOYEE_CSV, 'a', newline='', encoding='utf-8') as f:
            writer = csv.writer(f)
            writer.writerow([emp_id, name, dept, org_id, hobby]) # hobby の追加
        print('従業員を追加しました。')
    ```
